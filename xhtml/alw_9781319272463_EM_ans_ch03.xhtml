  • 3.37 Call the initial number chosen the “leading” number. All numbers form groups based off their leading number. Certainly, each leading number has an equal chance of being selected. Then, for each subsequent number, it is automatically chosen if its leading number is chosen. Which means all numbers (i.e., all groups) have an equal chance to be chosen. It is not a simple random sample because certain numbers can’t be chosen together; so not all samples are possible, only the preformed groups based off leading numbers.
