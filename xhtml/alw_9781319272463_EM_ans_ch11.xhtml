  • 11.49 It is highly unlikely to get a P-value so high; the random sample would have to produce an almost perfectly uniform result with an equal number (∼100) in each of the five groups. Most random samples will vary slightly, and getting a result so exact would be extremely rare. If the class had 2000 students, it is plausible that one (some) of the 2000 students might get an extremely rare result that provides an unusually high P-value.
